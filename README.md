# 💰 Expense Tracker (GUI)

A simple and user-friendly **Expense Tracker** built using Python and Tkinter.
This application helps users manage their income and expenses, track balance, and store transaction history.

---

## 🚀 Features

* 💵 Add **Income** and **Expense** transactions
* 📊 Automatic **balance calculation**
* 🗂️ Transaction history display
* 💾 Data stored in a **CSV file**
* ⚠️ Input validation
* 🧹 Clear all records option
* 🖥️ Simple and clean GUI

---

## 🛠️ Technologies Used

* Python 3
* Tkinter (GUI)
* CSV (for data storage)

## ▶️ How to Run

1. Install Python (3.x)
2. Download or clone this repository
3. Open terminal / command prompt
4. Run the program:

```bash id="l7n4y9"
python expense_tracker.py
```

---

## 🎮 How to Use

1. Select transaction type (**Income / Expense**)
2. Enter **amount**
3. Enter **category** (e.g., Food, Salary, Travel)
4. Click **Add Transaction**
5. View transactions in the list
6. Balance updates automatically
